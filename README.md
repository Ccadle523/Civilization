Civilization
============

文明模拟　第一版

这个版本所模拟的文明具有以下特征：
0，宇宙按照一定的几率产生新的文明；
1，所有文明具有一个共同的文明上限，越接近这个上限，文明发展速度越慢；
2，文明具有探索空间的能力，在探索空间范围内，对各星球是否存在文明可以有所感知；
3，文明的外交政策分为：主动攻击，帮助弱者，自我展示。不同的文明在这三项政策上会有不同；
3.1，主动攻击政策在发现新文明时会主动对其攻击，包括攻击过但没打死的文明，会持续攻击；
3.2，帮助弱者政策会在发现认识的文明比自己弱小时给予帮助；
3.3，自我展示则会在发现别的文明时主动告诉对方自己的存在；
4，当一个文明遭到攻击时，其性格决定了它是否反击；
5，当盟友遭到攻击时，文明会去帮助其发展，或者帮助其反击攻击者。
